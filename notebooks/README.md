# Notebooks

This folder organizes the RAG assignment notebooks.

## Sequence

1. `01_multimodal_vector_db.ipynb` — vector database initialization.
2. `02_advanced_rag_answer.ipynb` — advanced RAG answer generation.
3. `03a_basic_gradio_rag.ipynb` — basic Gradio RAG interface.
4. `03b_advanced_gradio_rag.ipynb` — advanced Gradio RAG interface.

Use the files in `data/raw/` as the sample knowledge base. Generated vector database files should be written to `data/vector_store/` and are ignored by Git.
